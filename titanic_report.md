## 3. 特征工程

### 3.1 缺失值处理
| 字段 | 处理策略 | 理由 |
|:---|:---|:---|
| `Age` | 中位数填充 | 分布相对对称 |
| `Embarked` | 众数填充 | 仅2条缺失 |
| `Fare` | 中位数填充 | 测试集可能缺失 |
| `Cabin` | 提取甲板字母 | 77%缺失，保留部分信息 |

### 3.2 特征构造

#### 头衔提取 (Title)
从姓名中提取社会头衔，反映性别、年龄、社会地位：
- Mr（成年男性）
- Mrs（已婚女性）
- Miss（未婚女性）
- Master（男孩）
- Rare（稀有头衔：Dr, Rev, Col等）

#### 家庭规模 (FamilySize)
```python
FamilySize = SibSp + Parch + 1
```
反映：
- 1 = 独自一人
- 2-4 = 小家庭（最优规模）
- 5+ = 大家庭（协调困难）

#### 是否独自一人 (IsAlone)
```python
IsAlone = (FamilySize == 1)
```

#### 年龄段 (AgeBin)
```
Child: 0-12岁
Teen: 12-20岁
Adult: 20-40岁
Middle: 40-60岁
Senior: 60+岁
```

#### 票价分段 (FareBin)
按四分位数划分：Low/Medium/High/VeryHigh

#### 舱房甲板 (CabinDeck)
提取舱房号首字母，反映甲板位置（A-G，T为顶层）

#### 人均票价 (FarePerPerson)
```python
FarePerPerson = Fare / FamilySize
```

| Embarked | One-Hot Encoding |
| Title | Label Encoding |
